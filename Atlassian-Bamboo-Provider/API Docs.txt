Discovery:
1. Does the application provide APIs for user management:
   Yes, Bamboo provides APIs for user management.

   https://docs.atlassian.com/atlassian-bamboo/REST/7.2.4/#d2e657

2. Are APIs for user management available in our plan.
   There are two plans. I think both the plans support user management because pricing
   is based on number of users.
    
   https://www.atlassian.com/software/bamboo/pricing

3. How does the API do authentication
   It provides two method:
   - Log into Bamboo manually
   - Use personal access token as a Bearer credential in an HTTP Authorisation header.

   https://developer.atlassian.com/server/bamboo/using-the-bamboo-rest-apis/
   https://confluence.atlassian.com/bamboo/personal-access-tokens-976779873.html
